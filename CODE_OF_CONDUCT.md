# Code of Conduct

This project follows the **Contributor Covenant**, version 2.1.

The full canonical text is maintained upstream at:
https://www.contributor-covenant.org/version/2/1/code_of_conduct/

By participating in this project — opening issues, submitting pull requests,
commenting in discussions, or interacting with other contributors — you agree
to abide by it.

## In short

- Be kind. Disagree on ideas, never on people.
- Welcome newcomers. Assume good faith. Ask questions before assigning blame.
- Keep critique focused on the work, not the person doing it.
- No harassment, no personal attacks, no demeaning comments.
- Maintainers will enforce these standards and may, at their discretion,
  remove comments, close issues or PRs, or block accounts that violate them.

## Reporting

If you witness or experience behaviour that violates this code, please report
it by **opening a private channel with the maintainer**:

- File a confidential GitHub Security Advisory on this repository (preferred,
  since GitHub keeps it private until disclosure), **or**
- Open a regular GitHub issue prefixed `[CONDUCT]` if the matter is not
  sensitive.

This is a small, personal-scale project — there is no formal moderation team.
Reports will be read and acted on by the repository owner. If the report
concerns the repository owner, please escalate through GitHub's own abuse
reporting at https://github.com/contact/report-abuse.

## Scope

This Code of Conduct applies within all project spaces — issue trackers, pull
requests, discussions, and any chat or social channels formally associated
with this repository — and also applies when an individual is officially
representing the project in public spaces.
